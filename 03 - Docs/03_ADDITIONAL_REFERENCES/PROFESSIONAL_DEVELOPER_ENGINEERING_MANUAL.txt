CRITICAL INFORMATION SUMMARY
================================================================================

THREE MOST IMPORTANT SIGNALS (Memorize These):
  1. I 72.3 (Cmd LockTable) - Prevents table movement during cutting
     ‚Üí When I 72.3 = 1: Table is LOCKED, oil pump blocked
     ‚Üí Safety: Without this, material could hit spinning blade
  
  2. I 72.7 (Disk Running) - Indicates cutting disk is spinning
     ‚Üí When I 72.7 = 1: Disk is CUTTING
     ‚Üí Safety: Table must stay locked to prevent collision
  
  3. I 73.7 (V/S Enable) - Master motion gate
     ‚Üí When I 73.7 = 0: ALL MOTION STOPS (emergency stop effect)
     ‚Üí Critical: Loss of this signal halts entire system

THREE MOST IMPORTANT FLAGS (Memorize These):
  1. F 0.0-0.3 - Program Mode Flags (mutually exclusive)
     ‚Üí Only ONE can be 1 at a time
     ‚Üí Determines operation: Manual, Translation, Passages, or Tilt
  
  2. F 5.0 - Motion Active (automatic tracking)
     ‚Üí Set when ANY motor energized
     ‚Üí Blocks mode changes and program starts (safety)
  
  3. F 1.6 - Stop All Motion (emergency)
     ‚Üí FORCES reset of all Q 6.0-6.5 contactors
     ‚Üí Triggered by safety conditions

MOST CRITICAL CODE SECTION:
  OB1 Section [8] - Table Lock (Lines 470-532)
  ‚Üí Most safety-critical code
  ‚Üí Q 8.7 (table lock solenoid) control
  ‚Üí Complex unlock conditions
  ‚Üí Prevents operator injury

MOST COMPLEX CODE:
  FB15 - Fast Command Mode (411 lines)
  ‚Üí Largest function block
  ‚Üí State machine with 8 options
  ‚Üí Handles quick-access menu

MOST COMPLEX PROGRAM:
  PB20 - Translation Program (112 lines)
  ‚Üí Position tracking across material
  ‚Üí Incremental repositioning
  ‚Üí Error recovery logic

DOCUMENT GLOSSARY
================================================================================

AWL:               Assembler/Step5 programming language for Siemens S5 PLC
DW:                Data Word (16-bit storage location)
F x.y:             Flag (internal bit for state management)
FB:                Function Block (subroutine with data)
I x.y:             Input bit (signal from field device)
OB1:               Organization Block 1 (main loop)
PB:                Program Block (subroutine)
Q x.y:             Output bit (command to field device)
T x:               Timer
VFD:               Variable Frequency Drive (inverter)
QW x:              Output Word (16-bit output)
Elbo:              Proportional joystick microcontroller
Soft-Start:        Star-Delta motor starting method
Ramp:              Smooth acceleration curve
Contactor:         Electromechanical relay for motor control
Solenoid:          Electromagnetic actuator (for table lock)
End Switch:        Limit switch preventing over-travel
Interlock:         Safety logic preventing dangerous conditions
